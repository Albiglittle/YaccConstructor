YaccConstructor

YaccConstructor is a modular tool for parser construction and grammars processing. Also it is a framework for research and development of parser generators, compilers compilers and other grammarware for .NET. YaccConstructor is mostly implemented in F#. Short name is YC.
YaccConstructor has been successfully used for a number of industrial projects, thus we can state that YC functionality is mature enough. Give it a try! We accept bug reports and are ready to make improvements on demand.
Main modules of YC.
  • YARD grammar specification language.
  • RNGLR parser generator.
  • GLL parser generator (in active development).
  • Platform for analysis of string-embedded languages.
