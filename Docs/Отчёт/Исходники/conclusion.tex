\section{Заключение.}

На данный момент работа продолжается, но уже есть следующие результаты:
\begin{itemize}
\item по однозначной LR-грамматике строится анализатор с линейной сложностью
\item по неоднозначной грамматике строится анализатор, возвращающий все возможные деревья вывода для данной входной цепочки
\item показана возможность поддержки регулярных выражений в правых частях правил
\end{itemize}

Задачи, требующие решения:
\begin{itemize}
\item Поддержка атрибутных грамматик. Построение анализатора.
\item Автоматическое востановление после ошибок.
\item Интеграция с YARD.
\end{itemize}
