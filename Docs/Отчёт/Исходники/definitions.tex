\section{Определения.}

Определения используемых терминов:
\begin{itemize}
\item GLR
\item EBNF: Extended Backus-Naur Form – расширенная нормальная форма Бэкуса–Наура, расширенная БНФ. 
\item ДКА
\item НДКА
\item Символ: общее название для терминала или нетерминал  при описании КА.   
\item Замыкание: q* = q$ \bigcup \{B\rightarrow.c | A \rightarrow a.Bb \in $q*$\} \bigcup \{x\stackrel{}{\rightarrow}.x | A\stackrel{}{\rightarrow} a.xb \in $q*$\}$
\item Функция goto: goto  q X = $\{A\stackrel{}{\rightarrow}aX.b | A\rightarrow a.Xb \in $q*$\}, $
\item LR-автомат
\item LR-ситуация
\end{itemize}
