\section{Заключение}

Полученные  результаты:
\begin{itemize}
\item проведён обзор инструментов построения анализаторов. Выяснено, что предпочтительным алгоритмом анализа является GLR алгоритм;
\item изучен рекурсивно-восходящий алгоритм анализа;
\item предложен прототип инструмента реализующего рекурсивно-восходящий алгоритм и применимого для работы с неоднозначными грамматиками;

\subsection{Свойства прототипа}

Предложенный прототип имеет следующие характеристики:
\begin{itemize}
\item по однозначной LR-грамматике строится анализатор с линейной сложностью;
\item по неоднозначной грамматике строится анализатор, возвращающий все возможные деревья вывода для данной входной цепочки;
\item показана возможность поддержки регулярных выражений в правых частях правил;
\end{itemize}
