\section{Постановка задачи}

В были сформулированы следующие требования к генератору синтаксических анализаторов:
\begin{itemize}
	\item Возможность работы с неоднозначными грамматиками. 
	\item Возможность воспринимать основные типы грамматик, встречающиеся в формальных описаниях языков. Как правило грамматики в EBNF.
	\item Поддержка атрибутных грамматик.
\end{itemize}

Цель данной работы:  разработать прототип инструмента, обладающего перечисленными выше возможностями.
