\subsection{Свойства прототипа}

\begin{itemize}
	\item по однозначной LR-грамматике строится анализатор с линейной сложностью;
	
	\item по неоднозначной грамматике строится анализатор, возвращающий все возможные деревья вывода для данной входной цепочки;
	
	\item показана возможность поддержки регулярных выражений в правых частях правил;
	
	\item пользователь имеет доступ к лесу вывода, что упрощает отладку;

	\item возможно дополнительные тестовые замеры;
